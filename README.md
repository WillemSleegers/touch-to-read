# Touch to Read

A speed reading app using RSVP (Rapid Serial Visual Presentation). Words are displayed one at a time at a fixed point — touch and hold to read, release to pause.

## Features

- **Touch-to-read interaction** — hold anywhere on screen to read, release to pause
- **Adjustable speed** — 100 to 1,000 words per minute
- **Smart punctuation pauses** — longer pauses at sentence endings and commas
- **ORP highlighting** — highlights the Optimal Recognition Point for faster word recognition
- **Multiple text sources** — paste text, upload `.txt`/`.md` files, or pick from samples
- **Settings persistence** — preferences saved to localStorage
- **Dark mode** — with system preference detection
- **Keyboard shortcuts** — Space (hold to read), arrow keys (skip words), R (restart), S (settings)
